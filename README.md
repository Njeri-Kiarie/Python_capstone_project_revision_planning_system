# Soma Planner
## Project Description
Soma Planner is a revision application that is designed to help students optimize, manage and schedule their study time especially during the exam period. By inputting their exam dates, subjects and difficulty weights, the system will calculate a personalized study timetable and generate a study plan based on content difficulty.

## Problem Statement
Students struggle to allocate the limited exam preparation time efficiently especially when they have subjects that they are struggling with as they juggle with the competing deadlines.

## Target Audience
University students preparing for multi-subject exam periods

## Planned Features
1. Subject management
2. Priority-based study planning
3. Automatic study plan generation
4. Quiz system and performance analysis
5. Study plan export to text file
6. Persistent Data storage
7. Dynamic sorting of revision priorites

## Technologies Used
1. Python: OOP, Exception Handling, File handling, Date and time manipulation and Randomization
2. Database technology: SQLite
3. Algorithms: Study priority, Sorting, Quiz scoring and Randomization
4. Data Structures: Dictionaries, Lists and Tuples
5. File Handling: Open, Write and Close files using with.
6. Exception Handling: try-except
7. Database Abstraction: db.execute()

## Data Source
No data needed

## Success Criteria
The project will be considered successful if students can register the subjects with regard to scheduled exams, generate prioritized study plans based on exam date allocation and subject difficulty, complete interactive quizzes with accurate scoring that modifies or updates the subject difficulty and have that stored persistently in the SQLite database. The system should also provide useful feedback and handle invalid inputs gracefully.
   
## Stretch Goals
* Email or Notification reminders when exams are approaching
* Calendar Integration
* Mobile or Web version

Given more development time, the system could be extended with user authentication, quiz history and analytics, calendar integration, reminder notifications and AI powered study recommendations. These enhancements would improve the overall learning experience.
